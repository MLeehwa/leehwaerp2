## 📋 확인해야 할 것

### 컨테이너 메뉴에서 확인:

1. **"컨테이너" 메뉴 클릭**
   - 왼쪽 사이드바에서 "컨테이너" 선택

2. **컨테이너 목록 확인**
   - `erp-mongodb` 컨테이너가 있는지 확인
   - `erp-mongo-express` 컨테이너가 있는지 확인

3. **상태 확인**
   - 상태가 "실행 중" 또는 "Running"인지 확인
   - "중지됨" 또는 "Stopped"이면 시작 버튼 클릭
